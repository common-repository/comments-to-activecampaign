The simplest way to collect leads from your comments to ActiveCampaign.

== Description ==

> #### Major features in Comments To ActiveCampaign include:
>
> * Collecting (syncing) leads when user leaves comments to yor post.
> * Future Updates will include adding tags, entering automation and more :).

[Check out the documentation](https://www.haktansuren.com/comments-to-activecampaign/?utm_medium=referral&utm_source=wordpress.org&utm_campaign=Comments+To+ActiveCampaign+Readme&utm_content=Documentation)

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `comments-to-activecampaign` folder to the `/wp-content/plugins/` directory via FTP
1. Activate the plugin through the 'Plugins' menu in WordPress
